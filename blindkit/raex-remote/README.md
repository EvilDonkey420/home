# raex-remote

A simple Serial-based RAEX remote transmitter for Arduino for use with a 433MHZ
transmitter. Choose a channel + remote combination and begin sending commands.

```
Channel: 169
Remote: 54062
Channel: 169, Remote: 54062
Action: D
Channel: 169, Remote: 54062
Action: U
Channel: 169, Remote: 54062
Action: s
Channel: 169, Remote: 54062
Action: u
```
